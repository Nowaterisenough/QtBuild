> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add repository-wide `vs2026` support for MSVC build flows with auto-detected Visual Studio paths.

**Architecture:** Extract the Visual Studio resolution logic into a shared PowerShell helper so it can be tested independently and reused by the MSVC workflows. Update the Qt 6 batch script to perform equivalent auto-detection locally, then wire both workflows and documentation to accept `2026` while preserving existing `2019/2022` behavior and artifact naming.

**Tech Stack:** Windows batch, PowerShell, GitHub Actions YAML, ripgrep

```powershell
function Resolve-VsInstall {
    param(
        [Parameter(Mandatory)] [string] $Version,
        [string] $PreferredEdition,
        [hashtable] $Roots
    )

    $supported = @{
        "2019" = ${env:ProgramFiles(x86)}
        "2022" = ${env:ProgramFiles}
        "2026" = ${env:ProgramFiles}
    }

    if (-not $supported.ContainsKey($Version)) {
        throw "Unsupported MSVC version: $Version"
    }

    $editions = @("Enterprise", "Professional", "Community", "BuildTools")
    if ($PreferredEdition) {
        $editions = @($PreferredEdition) + ($editions | Where-Object { $_ -ne $PreferredEdition })
    }

    $root = if ($Roots -and $Roots.ContainsKey($Version)) { $Roots[$Version] } else { Join-Path $supported[$Version] "Microsoft Visual Studio\$Version" }

    foreach ($edition in $editions) {
        $base = Join-Path $root $edition
        $vcvars = Join-Path $base "VC/Auxiliary/Build/vcvarsall.bat"
        $redist = Join-Path $base "VC/Redist/MSVC"
        if ((Test-Path $vcvars) -and (Test-Path $redist)) {
            return @{
                Version = $Version
                Edition = $edition
                VcvarsPath = $vcvars
                RedistPath = $redist
                VersionCode = "msvc${Version}_64"
            }
        }
    }

    throw "Visual Studio $Version not found under $root"
}
```

```bat
call :resolve_vs_install "%COMPILER_VERSION%" VCVARS_PATH || exit /b 1
call "%VCVARS_PATH%" amd64 || exit /b 1
goto :after_resolve

:resolve_vs_install
set "%~2="
set "VS_VERSION=%~1"
if "%VS_VERSION%"=="2019" (
    set "VS_ROOT=%ProgramFiles(x86)%\Microsoft Visual Studio\2019"
) else if "%VS_VERSION%"=="2022" (
    set "VS_ROOT=%ProgramFiles%\Microsoft Visual Studio\2022"
) else if "%VS_VERSION%"=="2026" (
    set "VS_ROOT=%ProgramFiles%\Microsoft Visual Studio\2026"
) else (
    echo ERROR: Unsupported MSVC version: %VS_VERSION%
    exit /b 1
)
for %%E in (Enterprise Professional Community BuildTools) do (
    if exist "%VS_ROOT%\%%E\VC\Auxiliary\Build\vcvarsall.bat" (
        set "%~2=%VS_ROOT%\%%E\VC\Auxiliary\Build\vcvarsall.bat"
        exit /b 0
    )
)
echo ERROR: Visual Studio %VS_VERSION% not found under %VS_ROOT%
exit /b 1

:after_resolve
```
